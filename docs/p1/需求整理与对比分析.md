# 基于问卷的需求整理结果

根据问卷调研结果，以下是校园互助服务平台的主要需求整理：

1. **用户注册与认证**  
   - 用户希望通过学号进行注册和认证，以确保平台用户均为校内学生，增加安全性。  
   - 用户对学号认证方式普遍接受，认为可以防止外校人员进入平台。  
   - （注：小部分用户有保护个人隐私、匿名或小号以提高自由度的需求）

2. **核心功能需求**  
   - 跑腿服务：包括快递代取、外卖代送、物品搬运等。  
   - 失物招领：方便用户发布和查找失物信息。  
   - 二手交易：支持学生之间的二手物品买卖。  
   - 学习互助：包括学习资料分享、课程辅导、疑问解答等。  
   - 生活帮忙：如咨询、求助、社区讨论等。  
   - 组队匹配：支持活动组队、课程组队、运动搭子等。

3. **交流与反馈**  
   - 用户希望在互助过程中能够与对方进行线上交流，以便更好地协调任务细节，部分用户希望平台同时有作为社交平台的需求。  
   - 用户希望平台能够提供评价反馈机制，以便了解互助双方的信誉和服务质量。  
   - 用户希望通过双方共同确认与平台作为中间人保留酬劳的方式确保任务完成与及时结算。

4. **酬劳与积分**  
   - 多数用户愿意接受积分作为酬劳，积分可用于兑换平台服务或商品。  
   - 部分用户希望可以直接获得现金或其他形式的酬劳。

5. **管理与维护**  
   - 用户希望平台能够管理言论、个人隐私、交易信誉等方面，确保平台氛围良好。  
   - 用户希望平台能够定期更新内容，保持活跃度，并提供便捷的搜索和推荐功能。

6. **收费方式**  
   - 用户对收费方式意见不一，部分用户希望完全免费，部分用户接受部分功能收费或自愿捐款。
